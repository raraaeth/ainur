# 🌸 Ainur Dashboard

> A personal life dashboard that combines Attendance, Finance, Crypto Portfolio, and Airdrop tracking into one beautiful Progressive Web App.

## ✨ About

Ainur Dashboard is a personal productivity dashboard built entirely with HTML, CSS, and Vanilla JavaScript.

It combines multiple aspects of everyday life into a single interface:

• Attendance
• Finance
• Crypto Portfolio
• Airdrop Tracker
• Smart Insight
• Planner
• Progressive Web App
